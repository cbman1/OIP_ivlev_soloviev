### Описание
Краулер ссылок и выделение описания и текста статей

Шаги:
1. **Сбора ссылок** (только на русскоязычные статьи) c сайта https://sibac.info,
2. **Выкачки содержимого** этих страниц (HTML с выборочным удалением ненужных блоков),
3. **Сохранения итоговых данных** в текстовые файлы.

Скрипты проекта:
- `scrape.py`: сбор ссылок на статьи и сохраняет их в `links.txt`.
- `main.py`: по списку в `links.txt` скачивает страницы, убирает лишние блоки и описания статьи на английском.

#### Дополнительные файлы

- **выкачка.zip**: результат в zip архиве
     - `выкачкаN.txt` (один файл на каждую страницу),
  - `выкачка_общая.txt` (объединённое содержимое),
- **index.txt** (соответствие &laquo;№ файла &rarr; ссылка&raquo;).
- **links.txt**: используемые ссылки
 



### Deployment Manual

1. Составление списка ссылок на статьи (файл `links.py`).
2. Выкачивания страниц по списку ссылок и очистки ненужных частей (файл `main.py`).

## 
python links.py

python main.py
