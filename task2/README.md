## **README к Заданию №2**

### Описание
Данный скрипт предназначен для обработки текстовых файлов, полученных из выкачки. Он выполняет следующие задачи:
1. **Чтение исходных файлов:**  
   Скрипт обрабатывает все файлы с расширением `.txt`, расположенные в директории `выкачка`.
2. **Токенизация и фильтрация:**  
   Из текста файлов извлекаются слова (латиница и кириллица). Извлечённые слова фильтруются:
   - Исключаются стоп-слова;
   - Пропускаются слова, состоящие только из английских букв;
   - Отбрасываются слова длиной ≤ 1 символ.
3. **Лемматизация:**  
   Применяется наивная лемматизация, при которой от токенов отсекаются известные суффиксы (если оставшаяся часть имеет длину не менее 3 символов).
4. **Сохранение результатов:**  
   Для каждого исходного файла создаётся отдельная подпапка (например, `выкачка1`, `выкачка2` и т.д.) внутри директории `выкачка_output`. В каждой из этих папок генерируются два файла:
   - `tokens.txt` — список уникальных токенов (по одному на строке);
   - `lemmas.txt` — группы токенов, сгруппированные по лемме, где токены для каждой леммы разделяются запятыми.

### Скрипты проекта
- `lemmas.py`: основной скрипт для обработки текстовых файлов. Он выполняет токенизацию, фильтрацию, лемматизацию и сохранение результатов в соответствующие выходные файлы.

#### Дополнительные файлы и структура директорий

- **Результаты работы:**
  - Папка `task2_output` — в ней создаются подпапки вида `выкачка1`, `выкачка2` и т.д.
    - Каждая подпапка содержит:
      - `tokens.txt` — файл с уникальными токенами (по одному токену в строке);
      - `lemmas.txt` — файл с леммами, где токены для каждой леммы разделяются запятыми.

### Deployment Manual

1. **Подготовка:**
   - Поместите исходные текстовые файлы в директорию `выкачка`.
2. **Запуск скрипта:**
   - Выполните команду:
     ```bash
     python lemmas.py
     ```
3. **Результат:**
   - После выполнения скрипта, результаты обработки будут сохранены в директории `task2_output`.

#### Предварительные требования

- Python 3.7+.
- Используются только стандартные библиотеки Python (без дополнительных зависимостей).
