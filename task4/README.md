## **README к Заданию №4**

### Описание

Реализация вычисления TF-IDF для текстовых документов. Основные задачи:

1. **Подсчет TF (Term Frequency):** Вычисление количества вхождений термина (raw count) в документе.
2. **Вычисление IDF (Inverse Document Frequency):** Оценка важности термина по формуле:
   
   IDF(t) = ln(N / df)
   
   где N — общее количество документов, а df — число документов, в которых термин встречается.
3. **Вычисление TF-IDF:** Комбинирование TF и IDF для оценки значимости термина в документе относительно всей коллекции документов:
   
   TF-IDF(t, d) = TF(t, d) * IDF(t)
4. **Сохранение результатов:** Для каждого документа создаются два файла:
   - `tokens_tf_idf.txt` – для терминов (tokens)
   - `lemmas_tf_idf.txt` – для лемм  
   Файлы сохраняются в папках:  
   `task4/task4_output/выкачкаN/`

### Файлы проекта (`task4/`)

* **`compute_tf_idf.py`**:
    * Читает текстовые файлы документов из `../task1/выкачка/выкачкаN.txt`.
    * Читает файлы `tokens.txt` и `lemmas.txt` из папки `../task2/task2_output/выкачкаN/`.
    * Выполняет очистку HTML-разметки и токенизацию текста.
    * Подсчитывает raw term frequency (TF) для терминов и лемм.
    * Вычисляет IDF для каждого термина и леммы по коллекции документов.
    * Рассчитывает TF-IDF как произведение TF на IDF.
    * Сохраняет результаты в файлы `tokens_tf_idf.txt` и `lemmas_tf_idf.txt` в папке `task4/task4_output/выкачкаN/`.
